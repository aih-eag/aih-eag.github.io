# What is Eaglercraft?
To put it simple, it is minecraft in the browser, for **free!**

---
# How to join the server?
To join the server, you must have a browser (Chrome, Microsoft Edge, Firefox, etc.)

Then you have to go to https://eaglercraft.com or [click here](https://eaglercraft.com).

After that, press the "Play Now!" button (which is orange)

Then there will be a screen with a small text written "Version" and under it choose the version "Release 1.8.8" or "Release 1.8.8 (WebAssembly)" and click "Play!"

Once you are done setting up your name and skin, you have to press "Multiplayer" then "Direct Connect" then "Join Shared World"

Then it will show up a box where you have to write the code from [here](https://rentry.co/aih-eag)
